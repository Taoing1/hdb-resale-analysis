## 第 2 页：项目背景

**为什么做这个项目？**
- 新加坡超 80% 人口居住在 HDB 组屋，转售市场活跃
- 房价受多重因素影响：房型、面积、屋龄、地段、楼层…
- 购房者面对海量数据，缺乏直观分析工具

**我们要解决什么问题？**
- 如何快速了解某个区域的房价全貌？
- 哪些因素对房价影响最大？
- 能否用模型预测一套房的价格？
- 预算有限的情况下，买哪里、买什么房型最划算？

**研究对象**
- 3 个相邻镇区：Punggol（榜鹅）、Sengkang（盛港）、Hougang（后港）
- 数据来源：data.gov.sg 官方 HDB 转售交易
- 时间范围：2020 年至今
- 记录数：**34,914 条**（清洗后）

## 第 7 页：价格预测 — 模型体系

**预测目标**：单价 price_per_sqm（新元/sqm）

**时间切分**：2020–2023 训练集 / 2024+ 测试集（禁止随机划分）

**特征工程（8 特征）**：
- 数值：面积、剩余租约、房龄、楼层中位、房型序数编码、MRT 距离、学校数量、交易年份
- 类别：镇区（OneHot 编码）

| 模型 | MAE (新元/sqm) | MAPE | R² |
|------|:---------:|:----:|:---:|
| Ridge Regression | ~S$520/sqm | ~7.7% | ~0.48 |
| Random Forest | ~S$700/sqm | ~10.0% | ~0.19 |

**Ridge 在时间切分中表现更优**：线性模型可通过 `year` 特征外推价格趋势，RF 无法外推时间趋势

**诊断**
- 分房型评估（6 类别：小户型/中户型/大户型/老旧/新近/MRT沿线）
- 预测值 vs 实际值散点图 + 残差分布诊断
- 5-fold 交叉验证

## 第 8 页：特征重要性

**模型特征重要性排名：**

| 排名 | 特征 | 重要性 |
|:---:|------|:---:|
| 1 | floor_area_sqm（面积） | ★★★★★ |
| 2 | flat_type_ordinal（房型编码） | ★★★★ |
| 3 | remaining_lease（剩余租约） | ★★★ |
| 4 | year（交易年份） | ★★★ |
| 5 | storey_mid（楼层） | ★★ |
| 6 | town（镇区） | ★★ |
| 7 | mrt_distance（MRT 距离） | ★ |
| 8 | schools_nearby（学校数量） | ★ |

**结论**：面积仍是最强预测因子，房型/镇区被面积部分替代

## 第 10 页：分析思考题（NEW）

**Q1：HDB 转售市场的「保值性」关键因素？**
- 面积—价格基本盘 → 剩余租约—时间折扣 → 镇区—地段溢价
- CAGR（年化复合增长率）量化 + 面积/房龄/镇区分组对比
- 约 200 字分析

**Q2：购房策略为何选该方向？**
- 4 策略逻辑：镇区偏离回归 / MRT 便利 / 长剩余租约 / 低总价入门
- 策略对应的数据逻辑 + Train/Test 验证
- 约 200 字分析

**Q3：模型预测能力的可靠性？**
- Ridge vs RF 时间切分对比
- 5-fold CV 验证
- 残差诊断 + 适用场景分析
- 约 200 字分析

## 第 11 页：策略验证 — 4 策略体系

**硬 2023/2024 时间切分**（防作弊）：2020–2023 策略形成期，2024+ 验证期

| 策略 | 逻辑 | 选择规则 |
|------|------|---------|
| 镇区偏离回归 | 买低于镇区均价的房产，等待回归 | 训练期价格偏离度最大 |
| MRT 便利 | 买靠近 MRT 的房产 | 训练期 MRT 距离最近 |
| 长剩余租约 | 买租约长的保值房产 | 训练期剩余租约最长 |
| 低总价入门 | 买低总价小户型 | 训练期总价最低 |

**7 项验证指标**：总涨幅 / CAGR / 价格波动 CV / 年度涨幅标准差 / 最大回撤 / 成交量 CV / 预算适配性

**输出**：策略组 vs 基准组对比表格 + 季度均价走势图 + 关键指标差异柱状图 + 策略反思（200+ 字）

## 第 12 页：技术亮点

1. **完整数据管线**：API → 缓存 → 自动清洗 → 3-sigma/分位数异常值处理
2. **双模型对比**：线性（可解释）+ 集成树（高精度），互补分析

6. **Town vs Price 偏离度**：三镇区均价偏离度 + CAGR 双重评估
7. **可扩展架构**：MRT/学校等图层已实现
8. **一键部署**：内置缓存数据，Streamlit Community Cloud 免费部署

## 第 14 页：项目挑战与解决

| 挑战 | 解决方案 |
|------|---------|
| CKAN API 分页上限（~5万条） | 按镇区分列请求，绕过全局限制，获取完整 2020-2026 数据 |
| 数据质量问题（缺失/异常） | 关键字段行删除 + 3-sigma 分组异常值剔除 + 分位数截尾 |
| 热力图无真实坐标 | 镇区聚类中心坐标 + 随机抖动（~300m 范围） |
| 时间切分下模型选择 | Ridge year 特征线性外推能力优于 RF 树模型外推 |
| 相似成交匹配过严 | 四级渐进式放宽（±10→±30sqm + 租约放宽 + 兜底策略） |
| Streamlit Cloud 无法访问 API | 内置缓存 Parquet 文件 + API 失败降级策略 |
| 面积 × 房型共线性 | Ridge L2 正则化 + RF 特征重要性聚合分析 |
| 多页面代码组织 | pages/ 目录 + importlib 动态导入 |
