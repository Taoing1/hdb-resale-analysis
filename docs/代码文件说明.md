# HDB 转售价格分析系统 — 代码文件说明

## 项目概述

基于 Streamlit 的新加坡 HDB 组屋转售价格分析与预测系统，覆盖 Punggol、Sengkang、Hougang 三个镇区。

## 运行方式

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用
streamlit run app.py

# 3. 浏览器访问
# http://localhost:8501
```

## 文件功能详解

### app.py — 主入口
- 设置 Streamlit 页面配置（标题、图标、宽屏布局）
- 通过 importlib 动态加载 5 个功能页面模块
- 侧边栏导航（radio 选择器）
- 调用 `load_data()` 加载数据 → 传递给当前页面的 `run(df)` 函数

### data/data_loader.py — 数据管线
- `fetch_hdb_data(force_refresh)`: 从 data.gov.sg API 分页获取数据，本地 Parquet 缓存，API 失败时降级使用缓存
- `clean_hdb_data(df)`: 12 步数据清洗流程（镇区筛选、日期解析、数值转换、剩余年限计算、房型标准化、楼层解析、单价计算、异常值剔除等）
- `load_data(force_refresh)`: 完整管线 fetch → clean

### data/hdb_cache.parquet — 数据缓存
- 内置 50,500 条原始 HDB 转售数据，清洗后目标镇区 9,713 条
- 确保 Streamlit Cloud 部署时无需网络也可运行

### pages/01_overview.py — 数据概览
- `_build_sidebar_filters()`: 9 项筛选控件（全选/清除快捷按钮）
- `_apply_filters()`: 多条件组合筛选
- `_render_stats_dashboard()`: 6 指标卡片 + 镇区分组统计表
- `_render_trend_chart()`: 月度均价走势（Plotly 折线图）
- `_render_volume_chart()`: 季度交易量（Plotly 柱状图）
- `_render_price_distribution()`: 价格分布直方图
- `_render_town_type_matrix()`: 镇区×房型 均价热力图
- `_render_data_table()`: 可配置列的交互式数据表 + CSV 导出

### pages/02_map.py — 地图可视化
- `_build_town_avg_map()`: 镇区均价 CircleMarker 地图 + 颜色图例
- `_build_heatmap()`: 房价热力图（含坐标抖动 + 镇区标签）
- `_build_temporal_map()`: 年份滑块地图 + 同比涨跌幅
- `_render_temporal_chart()`: 年度价格趋势线图 + 涨跌幅热力图
- `add_mrt_layer()` / `add_school_layer()` / `add_mall_layer()`: 叠加图层接口（已预留 API 接入代码注释）

### pages/03_factors.py — 影响因素分析
- `_analyze_flat_type()`: 箱线图 + 统计表 + ANOVA 检验 + 逐级溢价
- `_analyze_floor_area()`: 散点图 + OLS 趋势线 + Pearson 相关系数 + 单价分析
- `_analyze_lease()`: 分箱箱线图 + 散点图 + 各镇区折旧斜率 + 相关性
- `_analyze_town()`: 小提琴图 + 均价矩阵 + Kruskal-Wallis 检验 + 溢价分析
- `_analyze_interactions()`: 镇区×房型交互效应热力图
- `_analyze_partial_effects()`: 各因素 LOWESS 偏效应视图
- `_render_conclusions()`: R² 解释力综合结论表（单变量 + 联合模型）

### pages/04_prediction.py — 价格预测
- 特征工程：town / flat_type（OneHot）+ floor_area_sqm / remaining_lease / storey_mid / year（StandardScaler）
- `train_models()`: Ridge Regression + Random Forest（250 estimators, max_depth=18）
- 模型评估：MAE / RMSE / R² 指标卡片 + 柱状图对比
- 5-fold 交叉验证
- 残差诊断：残差分布直方图 + 残差 vs 预测值散点图
- 特征重要性：RF 基尼重要性（聚合）+ Ridge 标准化系数
- 交互式预测表单：用户输入 → 双模型预测 → 90% 预测区间 → 相似历史成交

### pages/05_strategy.py — 购房策略
- 约束输入：预算/房型/面积 + 三维度权重
- Train/Test 时间切分：前 75% 年份训练 → 后 25% 年份验证
- `_calc_appreciation()`: 训练期均价 → 测试期均价 CAGR 回测
- `_calc_stability()`: 变异系数 CV（小样本惩罚）
- `_calc_liquidity()`: 成交量归一化
- `_build_composite()`: 加权综合评分 + 排名
- 雷达图：Top 6 多维度对比
- 推荐输出：首选 + 备选 + 策略建议（自住/投资/短期/长期）
- `_render_affordability()`: 首付/月供/CPF 测算
- 政策分析接口（BSD/ABSD/HDB 资格）

### utils/helpers.py — 工具函数
- `TOWN_COLORS`: 三镇区配色映射
- `TOWN_COORDS`: 三镇区经纬度坐标
- `fmt_price()`: 新元 金额格式化（K/M 单位）
- `fmt_pct()`: 百分比格式化
- `get_model_metrics()`: 计算 MAE / RMSE / R²
